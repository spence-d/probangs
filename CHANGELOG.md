# Change Log

## 0.1.0 - 2026-07-04
### Added
- Functioning search bar, settings, and links widget.
- Support for basic syntaxes: `!b ![b]@[site] b![k=]v[@site]`
- Support for multiple bangs in a query
- Nonvolatile settings: autofocus, history, suggestions, default, lucky
- List of bangs with aliases, param tags, default param, suggestion urls, lucky
urls
- Live suggestions for bangs and search queries
- Scrolling history of past searches
